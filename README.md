# banker-algorithm-simulator-
This project is a Python-based Banker’s Algorithm Simulator designed to demonstrate deadlock avoidance in Operating Systems. It provides a graphical user interface using CustomTkinter, allowing users to input process allocation, maximum resource requirements, and available resources. The system calculates the Need Matrix and determines whether the system is in a safe or unsafe state. If safe, it also generates a safe sequence of process execution. The project follows a structured layered architecture including input validation, processing logic, and output visualization. It helps students understand operating system concepts in a practical and visual way. The interface is designed with a modern Windows 11 style UI for better user experience and clarity.
## Description
This project implements Banker’s Algorithm using Python GUI (CustomTkinter) to demonstrate deadlock avoidance in Operating Systems.

## Features
- Safe/Unsafe detection
- Safe sequence generation
- Modern GUI interface
- Table-based input system

## Tech
- Python
- CustomTkinter
